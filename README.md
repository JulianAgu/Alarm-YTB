# 🚀 Alarm YTB: Smart YouTube Scheduler
**The ultimate tool to automate your YouTube viewing and time management.**

Alarm YTB is a professional Edge extension that ensures you never miss a study session, a health break, or the most anticipated YouTube premieres.

You can install the official version directly from the Microsoft Edge Add-ons store:
👉 **[Get Alarm YTB on Microsoft Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/alarma-ytb/aapfcdolacgeejhbnmbmppgohockliab)**

## ✨ Why Alarm YTB?

* **🎥 Premiere & Live Stream Ready:** Don't just set a reminder—automate the opening. Set the timer, paste the URL, and be the first one in the chat.
* **⏰ Precision Timing:** Supports exact minutes and decimals for high-accuracy scheduling.
* **🕒 Hourly Sync (Every :00):** Stay on track with automatic triggers at the start of every hour.
* **🔊 Personalized Alerts:** Use any video or Short as your alarm sound. No more boring system beeps.
* **🟢 Pro Visual Feedback:** * **Dynamic UI:** Header and clock change color based on status (Green/Active vs Red/Inactive).
    * **Pulse Animation:** A subtle glow that tells you the timer is running at a glance.
* **💾 Persistent Memory:** Your settings are safe. Even if you restart your browser, your last URL and time are saved.

---

## 🛠 Tech Stack
Built with modern web standards for speed and reliability:
- **JavaScript (ES6+):** Core logic and timing.
- **Chrome/Edge Extension API:** Leveraging `alarms` for background persistence and `storage` for user data.
- **CSS3 Modern UI:** Neumorphic design elements, CSS Grid, and custom animations.
